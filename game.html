Some games I worked on...

YuanGu Project

(The game is only available in Chinese!) A cute RPG game where you control plant-creatures to fight enemies! A project that I worked on as the lead programmer for about a year with 5 other developers.



Ragnarok TD

A Bloon's TD replica game that I worked on with 10+ students for 3 months. As a programmer, I helped on creating a lot of the tower systems and some of the UI elements.



Treat Team

A casual puzzle game that is currently under open beta. I am currently leading the development of the new base camp feature which will allow players to build buildings, meet new animal friends and interact with them while they venture into the forest. I also worked on some of the puzzle level designs.



A Maze In

A multiplayer maze/puzzle game that I completed with 3 other developers in a month. Players need to traverse through the maze while answering quiz questions and avoiding monsters. They can also utilize a set of unique skills such as teleport, heal, and freeze to distract the opponent or help themselves.



Smash!

A rogue-like adventure mobile game that is currently under development! I started this project myself and am now working with 5 other developers to bring it to light!
